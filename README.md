This project is a Machine Learning based Diet Recommendation System that generates personalized diet plans according to user health credentials and inputs.

## Features
- Personalized diet recommendations
- Machine Learning based prediction system
- Integrated with web application
- User-friendly backend API support
- Fast and efficient recommendation generation

## Machine Learning Models Used
- Random Forest
- Logistic Regression

## Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy

## Project Workflow
1. User enters health and personal information on the website.
2. The web application sends the data to the backend.
3. The trained Machine Learning model processes the input.
4. The system generates a personalized diet plan recommendation.
